Welcome! This repository is a hobby project dedicated to translating the fantastic **[Data Science for Ecologists and Environmental Scientists](https://ourcodingclub.github.io/course)** course by the [Coding Club](https://ourcodingclub.github.io/) from its original **R** implementation into **Python**.



## 🌲 Motivation



The original Coding Club course is an incredible resource designed to help ecologists, environmental scientists, and quantitative biology enthusiasts overcome "code fear" and statistics anxiety. While R is a staple in ecology, Python has grown immensely as a powerhouse for data science, machine learning, and geospatial analysis. 



This repository aims to bridge the gap by providing Python equivalents (using libraries like `pandas`, `matplotlib`, `seaborn`, `statsmodels`, and `geopandas`) to the original R workflows. It's meant as a supplementary learning tool for those who prefer Python or want to learn how to translate concepts between the two languages.



## 📚 Course Progress & Content



The original course covers several streams. Here is the progress of translating the core tutorials into Python:



### Stream 1: Data Manipulation & Visualisation

- [ ] Basics of functional and object-oriented programming

- [ ] Data manipulation and organisation (translating `dplyr`/`tidyr` to `pandas`)

- [ ] Data visualisation and graphics (translating `ggplot2` to `matplotlib`/`seaborn`)



### Stream 2: Modelling & Statistics

- [ ] The linear model (using `statsmodels` or `scikit-learn`)

- [ ] Hierarchical linear models / mixed-effects models

- [ ] Bayesian statistics



### Stream 3: Advanced Skills

- [ ] Geospatial analysis (using `geopandas`, `rasterio`, etc.)

- [ ] Development of workflows for quantitative analysis

- [ ] Version control and collaborative coding

## 💻 How to Use This Repository



To run the code in this repository, you will need Python installed (preferably 3.8 or higher) and a few standard data science libraries.



1. **Clone the repository:**

   ```bash

   git clone [https://github.com/malgocie/data-science-for-ecologists-and-enviromental-scientists.git](https://github.com/malgocie/data-science-for-ecologists-and-enviromental-scientists.git)

   cd data-science-for-ecologists-and-enviromental-scientists



2. **Set up a virtual environment (optional but recommended):**

   ```bash

   python -m venv env

   source env/bin/activate  # On Windows use: env\Scripts\activate

   

3. **Install the required packages:**

   ``` bash

    pip install -r requirements.txt

   

4. **Explore the tutorials:**

   Most tutorials are written as Jupyter Notebooks (.ipynb) so you can read the explanations and run the code block by block.

   ``` bash

   jupyter notebook
